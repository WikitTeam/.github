# WikitTeam

**WikitTeam** is a technical team focused on Wikidot and similar wiki platforms, dedicated to providing users with efficient management, search, and migration solutions.

---

## Project Goals

- **Migration Support**: Tools to migrate Wikidot content to modern platforms like ProjectWikit.  
- **Feature Extensions**: Develop various wiki plugins and management tools to improve site usability and administration efficiency.  
- **Data Applications**: Enable users to access and utilize wiki data more flexibly, supporting custom data applications.

---

## Core Projects & Tools

### ProjectWikit
- A modern platform compatible with Wikidot syntax (FTML)  
- Supports migrating data from existing Wikidot sites

### Wikit Search
- Efficient solution for searching Wikidot sites  
- Supports multiple search modes:
  - Page content keyword search
  - Author search
  - Tag search
  - Date range search
  - And more

### Wikit GraphQL
- Efficiently indexes all wiki, page, and user data  
- Allows users to create custom data applications and queries

### Other Wiki Extensions & Management Plugins
- Data scraping, backup, and statistics tools  
- Custom templates, styles, and site management plugins  
- Focused on improving management efficiency and content usability
